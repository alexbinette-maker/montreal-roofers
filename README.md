# Montreal Roofers Website

Professional roofing services website for Montreal Roofers, serving Pointe-Claire and the West Island.

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Professional Layout** - Modern, clean design with clear call-to-actions
- **Services Showcase** - Highlights all roofing services offered
- **Testimonials** - Social proof section with customer reviews
- **Contact Forms** - Easy-to-use contact and quote request forms
- **Mobile Menu** - Responsive navigation for smaller screens
- **SEO Optimized** - Meta tags and structured content for search engines

## Services

- Roof Repairs
- Roof Replacement
- Emergency Services (24/7)
- Roof Inspections
- Commercial Roofing
- Snow Removal
